# GPT-image2 分镜受控替换提示词

用于 Stage 2。默认必须使用 GPT-image2 生成新分镜图；如果用户已经提供外部 GPT-image2 生成后的新分镜图，则直接把该图作为 Stage 2 产物校验，不要重复生成。

## 动态填充规则

生成替换提示词前，必须已经确认产品、人物、场景三项分别是“提供素材替换”还是“沿用原样”。如果用户说要替换某项但没有提供对应素材，停止并向用户索要素材，不要生成替换提示词。

根据用户实际提供的素材改写第一句：

- 产品图存在：`只替换分镜图中的产品为我上传的产品图。`
- 场景参考存在：`只替换分镜图中的场景为我上传的场景参考。`
- 人物参考存在：`只替换分镜图中的人物为我上传的人物参考。`
- 某项不存在：写明该项保持原样。

如果产品、场景、人物都确认沿用原样，默认不调用 GPT-image2，直接沿用原分镜图。

如果没有 GPT-image2 或生图 API 可调用，不能生成或伪造新分镜图；只输出请求包和替换范围校验图，并停止等待。

如果用户提供了外部生成的新分镜图，必须校验：

1. 新分镜格子数量。
2. 原分镜格子数量。
3. 新旧格子数量是否一致。
4. 替换项是否符合约定。
5. 未提供的人物/场景是否尽量沿用原样。

新旧格子数量不一致时，不要自动拒绝，但必须明确标记 Stage 4 需要做动作映射，并等待用户确认。

## Prompt 模板

```text
请基于我上传的原分镜图进行受控替换。

只替换分镜图中的产品为我上传的XXX，场景为我上传的XXX，人物为我上传的XXX。

用户没有提供的元素，必须保持原分镜图中的原样内容。

其余内容必须与原分镜图保持完全一致。

禁止重新设计画面构图。
禁止修改镜头角度。
禁止修改运镜。
禁止修改景别。
禁止修改人物动作。
禁止修改画面节奏。
禁止修改光影关系。
禁止修改分镜结构。

保持与原分镜图完全一致的格子数量。

禁止新增、删除、合并、重排分镜格子。

人物动作、人物姿势、人物位置必须严格对齐原分镜图对应格子。

最终结果需实现与原分镜图的1:1视觉对齐。
```

## 输出校验

生成后检查：

1. 分镜格子数量是否一致。
2. 分镜顺序是否一致。
3. 构图、镜头角度、景别是否被改动。
4. 人物动作、姿势、位置是否严格对齐。
5. 未提供的产品/人物/场景是否保持原样。
6. 是否出现新增人物、产品、道具、文字、logo、字幕或水印。
